# Инструкция пользователя

## Вход в систему
- Вариант 1: логин/пароль.
- Вариант 2: сканирование персонального QR-кода сотрудника.

## Работа в модуле «Склад»
1. Создайте поступление.
2. Добавьте товары (код, наименование, количество, цена).
3. Проведите документ — остатки увеличатся.

## Работа в модуле «Лавка»
1. Создайте продажу.
2. Сканируйте QR-код товара или выберите товар вручную.
3. Укажите количество.
4. Проведите продажу — остатки уменьшатся.

## Отчёты
- Формируйте оборотно-сальдовую ведомость за период.
- Проверяйте остатки на конец периода.

## Важное
- Если день закрыт супервизором/администратором, продажи за этот день изменять нельзя.
