# Инструкция администратора

## Задачи администратора
- Создание пользователей и назначение ролей.
- Блокировка/разблокировка учётных записей.
- Контроль прав доступа к разделам.
- Контроль закрытия операционного дня.
- Настройка резервного копирования SQLite-файла на SFTP/SCP.

## Роли
- `shop_worker`: Лавка (чтение/запись)
- `warehouse_worker`: Лавка (чтение), Склад (чтение/запись)
- `supervisor`: Лавка+Склад (чтение/запись), закрытие дня
- `admin`: полный доступ

## Рекомендации по backup
1. Останавливать приложение перед backup (или использовать snapshot WAL).
2. Копировать `church_warden.db` и сопутствующие `-wal`, `-shm` при наличии.
3. Пример копирования:
```bash
scp church_warden.db user@backup-host:/opt/backups/church/
```
или
```bash
sftp user@backup-host
put church_warden.db /opt/backups/church/
```
4. Хранить не менее 30 ежедневных копий.
5. Ежемесячно проверять восстановление на тестовом стенде.
