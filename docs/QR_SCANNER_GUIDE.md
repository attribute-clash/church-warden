# QR-сканер для Linux: рекомендации и подключение

## Рекомендуемые модели
1. **Datalogic QuickScan QW2520** (2D, USB, HID Keyboard) — надёжный корпоративный вариант.
2. **Zebra DS2208** (2D, USB) — широко распространён, стабильная работа в Linux как HID.
3. **Honeywell Voyager 1470g** (2D, USB) — качественный сканер для интенсивного использования.

## Почему именно такие модели
- Поддержка 2D QR-кодов.
- Режим HID (эмуляция клавиатуры) — обычно драйверы не требуются.
- Поддержка USB-подключения «plug-and-play» в Linux.

## Подключение к ноутбуку (Linux)
1. Подключите сканер по USB.
2. Проверьте, что устройство определилось:
```bash
lsusb
```
3. Проверьте события ввода:
```bash
sudo evtest
```
или
```bash
sudo libinput debug-events
```
4. Откройте любой текстовый редактор и отсканируйте тестовый QR — строка должна вводиться как клавиатурный текст.

## Драйверы
- Для режима HID отдельные драйверы чаще всего **не нужны**.
- Для фирменных утилит конфигурации берите ПО только с официального сайта производителя:
  - Datalogic: https://www.datalogic.com/
  - Zebra: https://www.zebra.com/
  - Honeywell: https://automation.honeywell.com/

## Что настроить на сканере
- Суффикс `Enter` после сканирования (удобно для форм ввода).
- Префиксы/суффиксы только при необходимости.
- Проверить тип кодировки UTF-8/ASCII в документации модели.
