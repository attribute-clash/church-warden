# Church Warden (Rust)

Локальная система учёта продаж и поступлений для церковной лавки.

## Что реализовано
- Модули: `auth`, `shop`, `warehouse`, `reports`, `admin`.
- Локальная БД SQLite.
- Вход по логину/паролю и QR-токену сотрудника.
- Продажи, поступления, отчёт по оборотам, закрытие дня.
- Юнит-тесты и интеграционные тесты для каждого модуля.

## Быстрый старт
```bash
cargo run
```

## Тесты
```bash
cargo test
```
