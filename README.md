# Church Warden (Rust + GUI)

Локальная система учёта продаж и поступлений для церковной лавки с полноценным GUI:
- левая панель разделов (по ролям);
- табличные экраны модулей;
- вход по логину/паролю и QR;
- резервное копирование по SCP/SFTP.

## Реализованные модули
- `auth` — пользователи, пароли (argon2), QR-вход.
- `shop` — продажи и списание остатков.
- `warehouse` — поступления и приход товаров.
- `reports` — оборотно-сальдовый отчёт и закрытие дня.
- `admin` — роли, видимость разделов.
- `backup` — резервное копирование файла БД по SSH (SCP/SFTP).

## Запуск GUI
```bash
cargo run --release
```

## Тестирование
```bash
cargo test
```
